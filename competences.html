Compétences

* Les jauges indiquent le niveau attendu d'un étudiant de deuxième année de BUT

Réseau

Configuration réseau : 80%
Lors de nombreux TD et TP, j'ai pu me familiariser avec l'utilisation et la configuration des différents appareils utilisés dans un réseau. L'utilisation du logiciel Cisco Packet Tracer m'a été très utile pour apprendre les différentes facettes des réseaux comme par exemple les protocoles (tels que DHCP, HTTP, Ethernet), l'adressage ou la configuration poussée de routeurs ou de switches.

Gestion de bases de données : 70%
J'ai pu apprendre à manipuler des bases de données avec le logiciel MySQL. J'ai pu apprendre à créer des tables, à les remplir et à les modifier. J'ai aussi appris à faire des requêtes SQL pour récupérer des informations dans ces tables de différentes manières (boucle, fonction, requêtes imbriquées ...).

Administration système : 80%
Durant les différents TD et TP j'ai appris à utiliser des logiciels permettant l'administration et la gestion de systèmes Linux et Windows. J'ai par exemple vu comment utiliser l'Active Directory pour gérer les utilisateurs et les groupes sur un serveur Windows ou le fonctionnement d'un hyperviseur pour gérer des machines virtuelles.

Programmation

Python : 80%
Durant mes années de lycée, j'ai pu m'initier à la programmation sur le langage Python. Les nombreux TD et TP dans le BUT m'ont permis de consolider ces connaissances et de m'améliorer dans la méthode de programmation.

Bash : 70%
Grâce à des TP et des TD, j'ai pu apprendre les différentes commandes utiles pour utiliser le système d'exploitation Linux, de plus j'ai pu apprendre le langage Bash permettant de faire des scripts et d'automatiser certaines tâches sur Linux.

HTML/CSS/PHP : 75%
La création de ce portfolio ainsi que les différents projets m'ont aidé à progresser dans la programmation avec ces langages. J'ai pu apprendre à utiliser le langage PHP pour créer des pages dynamiques et à utiliser des bases de données SQL pour stocker des informations. De plus, j'ai appris à faire communiquer les 2 langages pour récupérer des informations dans la base de données et les afficher sur une page web.

Télécommunications

Numérisation : 70%
Pendant ma formation, j'ai appris à numériser un signal analogique en voyant les différentes étapes nécessaires. J'ai par exemple vu comment fonctionnait l'échantillonnage et la quantification d'un signal analogique à l'aide d'un oscilloscope et comment le convertir en un signal lisible pour un ordinateur.

Fibre optique : 70%
Pendant ces deux années de formations, j'ai appris à utiliser des fibres optiques. J'ai vu le fonctionnement des fibres monomode et multimode ainsi que leurs différences physique.

Téléphonie

Configuration d'un réseau de téléphonie : 70%
Grâce aux cours dispensés dans le BUT j'ai pu me familiariser avec les différents équipements nécessaires à la mise en place d'un réseau de téléphonie. Nous avons notamment vu comment configurer des téléphones de différents types (SIP, numérique, softphone ...) et comment les faire communiquer en mettant en place un call server.
