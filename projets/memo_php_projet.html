Contexte

J'ai réalisé ce projet dans le cadre de mon BUT afin de découvrir le langage PHP.

Objectifs

L'objectif de ce projet était de réaliser un jeu de memory en PHP pour apprendre ce langage et le fonctionnement des cookies sur les navigateurs.

Mission

Ma mission était de suivre l'ossature donné par le professeur en programmant les fonctions demandées grâce à la documentation PHP et mes connaissances développé au premier semestre. A la fin le jeu devait être totalement jouable sur navigateur avec une interface graphique fait en CSS.

Résultats

Le projet est entièrement fonctionnel et jouable sur navigateur. Vous pouvez le tester avec le lien présent dans l'encart Document.
